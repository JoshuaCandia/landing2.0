# Wonkai Landing Page

Landing page moderna y atractiva para Wonkai, una software factory especializada en desarrollo de software personalizado con inteligencia artificial.

## 🚀 Características

- **Diseño moderno**: Estilo futurista con modo oscuro y efectos visuales
- **Responsive**: Optimizado para desktop y mobile
- **Animaciones suaves**: Implementadas con Framer Motion
- **Performance**: Construido con Astro para máxima velocidad
- **Tipografía moderna**: Inter y Poppins para una experiencia visual premium

## 🛠️ Tecnologías

- [Astro](https://astro.build/) - Framework web moderno
- [Tailwind CSS](https://tailwindcss.com/) - Framework de CSS utility-first
- [Framer Motion](https://www.framer.com/motion/) - Biblioteca de animaciones
- [React](https://reactjs.org/) - Para componentes interactivos
- [TypeScript](https://www.typescriptlang.org/) - Tipado estático

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Construir para producción
npm run build

# Preview de la build
npm run preview
```

## 🎨 Paleta de colores

- **Fondo**: #0B0B14
- **Principal**: #8B5CF6 (violeta)
- **Secundario**: #22D3EE (celeste)
- **Texto**: #E5E5E5 (blanco/gris claro)

## 📱 Secciones

1. **Hero Section** - Presentación principal con CTA
2. **Sobre nosotros** - Filosofía y estadísticas de la empresa
3. **Proyectos** - Showcase de trabajos destacados
4. **Servicios** - Cards con servicios ofrecidos
5. **CTA Principal** - Llamada a la acción para contacto
6. **Footer** - Información de contacto y redes sociales

## 🚀 Despliegue

El proyecto está configurado para generar un sitio estático que puede desplegarse en cualquier hosting estático como:

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages
